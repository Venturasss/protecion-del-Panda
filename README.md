Proteccion del Panda
Descripción
Proyecto web educativo enfocado en la conservación del panda gigante (Ailuropoda melanoleuca). 
Este sitio web presenta información relevante sobre su hábitat natural en China y sus hábitos a
limenticios específicos, priorizando una experiencia de usuario (UX) limpia y un rendimiento web optimizado.

Tecnologías utilizadas
Framework: Astro (Arquitectura de islas para alto rendimiento).
Frontend: React.js, Tailwind CSS.
Optimización: Web Performance (Lighthouse) y estrategias de carga rápida.
Gestión de código: Git y GitHub.

Características principales
Diseño Responsivo: Interfaz adaptada a dispositivos móviles y escritorio.
Rendimiento: Migrado a Astro para reducir los tiempos de carga inicial, logrando una navegación fluida.
Código Limpio: Estructura modular de componentes para facilitar el mantenimiento y escalabilidad.
Gestión Profesional: Control de versiones mediante Git, garantizando un flujo de trabajo seguro y organizado.
